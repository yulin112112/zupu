电子族谱使用教程

1. 简介

电子族谱是一个基于浏览器的族谱管理工具，可以帮助您创建、编辑和保存家族树。它支持添加多媒体内容（图片、视频、音频）到族谱成员信息中，并可以将整个族谱导出为ZIP文件进行备份或分享。

2. 开始使用

2.1 添加根节点

  1. 打开电子族谱网页
  2. 点击顶部的添加根节点按钮
  3. 在弹出的对话框中填写姓名、性别、配偶姓名、民族、出生日期和死亡日期等信息
  4. 点击确定按钮完成添加

3. 管理族谱节点

3.1 添加父节点

  1. 右键点击要添加父节点的成员
  2. 在右键菜单中选择添加父节点
  3. 填写相关信息后点击确定

3.2 添加子节点

  1. 左键点击或右键点击要添加子节点的成员
  2. 点击添加子节点按钮或在右键菜单中选择添加子节点
  3. 填写相关信息后点击确定

3.3 编辑节点

  1. 左键点击或右键点击要编辑的成员
  2. 点击编辑信息按钮或在右键菜单中选择编辑节点
  3. 修改信息后点击确定

3.4 删除节点

  1. 右键点击要删除的成员
  2. 在右键菜单中选择删除节点
  3. 确认删除操作（注意：删除节点会同时删除其所有后代）

4. 管理成员信息

4.1 添加头像

  1. 左键点击要添加头像的成员
  2. 在右侧信息面板中点击上传头像按钮
  3. 选择图片文件后点击打开

4.2 添加多媒体内容

  1. 左键点击成员，在右侧信息面板中找到生平介绍、身后事或墓地部分
  2. 点击添加条目按钮创建新条目
  3. 在条目中填写标题和文字内容
  4. 点击添加图片/音视频按钮上传多媒体文件

注意：支持的图片格式包括 JPG、PNG、GIF 等；支持的视频格式包括 MP4、WebM 等；支持的音频格式包括 MP3、WAV 等。

5. 保存与导出

5.1 保存族谱

  1. 点击顶部的保存按钮
  2. 族谱数据将保存到浏览器的本地存储中

5.2 导出族谱

  1. 点击顶部的导出按钮
  2. 系统会自动生成包含族谱数据和多媒体文件的 ZIP 文件
  3. 浏览器会自动下载该 ZIP 文件

5.3 导入族谱

  1. 点击顶部的导入按钮
  2. 选择之前导出的 ZIP 文件
  3. 系统会自动加载族谱数据和多媒体文件

6. 其他功能

6.1 编辑代数信息

在族谱图中，每一代的标识文字（如"第1代"）可以直接点击进行编辑。

6.2 清除族谱

点击顶部的清除族谱按钮可以删除所有族谱数据，请谨慎操作。

7. 注意事项

  • 族谱数据保存在浏览器本地存储中，清除浏览器数据可能会导致数据丢失
  • 建议定期使用导出功能备份族谱数据
  • 导入功能仅支持本系统导出的 ZIP 文件
  • 多媒体文件以 Base64 格式保存在族谱数据中，过多的多媒体内容可能会导致文件体积较大
